Install ZTrade


ZTrade is a multi-platform application, you can use the Desktop and Mobile version or only one of them.
For the best user experience, MetricTrade suggests using both a Desktop and a Mobile version of the app.

Install ZTrade by selecting your device/devices:


Windows 11 / 10

Please click here to get the application from the Microsoft Store and pin it to the taskbar.
Or from the Microsoft Store application on your device, search "ZTrade Plus" and select "Install".
You can launch the application by clicking its icon or by searching "ZTrade Plus".

macOS - 10.15 or later

Please click here to get it from the Mac App Store.
Or from the Mac App Store application on your device, search "ZTrade Plus" and select "Install".
Open the Application by clicking on its icon in your device application list.

iOS - iPhone / iPad

Please click here to get it from the App Store.
Or from the App Store application on your device, search "ZTrade Plus" and install it.
Open the Application by clicking on its icon in your device application list.

Android- Phone / Tablet

Please click here to get it from the Google Store.
Or from the Google Play Store application on your device, search "ZTrade Plus" and install it.
Or from the Google Play Store application on your device, search "ZTrade Plus" and install it.
Open the Application by clicking on its icon in your device application list.

Windows 8/7/XP/Vista

Microsoft Store is not available on Windows 8, 7, XP and Vista.
The ZTrade application can run on your computer using Java.
To run ZTrade on Windows using Java follow the below steps:

1. If you haven't installed Java on your PC download it from here https://www.java.com/
2. Download ZTrade from here
3. If you are using Chrome click Keep in the file confirmation messages and then click on the zplusi.jnlp file.
4. Right -click the file and then select Open from the context menu.

macOS - 10.14 or earlier

ZTrade for Mac App Store is not available on macOS 10.14 or earlier.
The ZTrade application can run on your computer using Java.
To run ZTrade on macOS using Java follow the below steps:

1. If you haven't installed Java on your Mac download it from here https://www.java.com/
2. Download ZTrade from here
3. Locate the file zplus.jnlp in Finder.
4. With right-click or by holding the Control key click on the file and then select Open from the menu.
